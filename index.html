ทำไมถึงตัดสินใจเรียนวิศวกรรมเครือข่ายและความมั่นคงไซเบอร์

ตอนเด็กผมมีไอดอลคนนึงที่ทำเกี่ยวกับการเขียนcodeเขาได้โชว์การทำเกมให้คนอื่นดูตอนนั้นผมจึงเริ่มสนใจและได้ไปค้นคว้าว่าRoblox Studioสามารถใช้ทำเกมได้ผมจึงได้เริ่มศึกษาlua,blenderลองผิดลองถูกไปหลายอย่างจนผมได้แมพมาจุดนั่นทำให้ผมมีความสนใจในการเขียนcode ต่อมาผมได้ลองลงโปรแกรมต่างๆลงเครื่องนั้นทำให้เครื่องผมติดไวรัสเข้า ผมจึงเริ่มค้นหาข้อมูลจนรู้ว่าผมไปลงโปรแกรมต่างๆจากเว็บที่แฝงไวรัสไว้ทำให้เครื่องผมมีโอกาสติดไวรัสเหตุการณ์นี้ทำให้ผมรู้สึกอยากปกป้องคนในครอบครัวไม่ให้เจอปัญหาแบบเดียวกัน ผมจึงหันมาสนใจเรื่อง NetworkCybersecurity และ Coding อย่างมากผมจึงคิดว่าวิศวกรรมเครือข่ายและความมั่นคงไซเบอร์ นี่แหละเป็นเส้นทางที่เหมาะสมที่สุดสำหรับผม และเมื่อเรียนจบแล้ว ผมจะทำงานในสาย Network Security อย่างมีประสิทธิภาพ และเป็นประโยชน์ต่อสังคม
